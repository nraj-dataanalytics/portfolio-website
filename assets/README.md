# Assets Folder

This folder contains your resume and other downloadable assets.

## How to Add Your Resume:

1. Save your resume as `resume.pdf` in this folder
2. The website will automatically link to it from:
   - Hero section "Download Resume" button
   - Resume section "Download Full Resume" button

## Supported File Types:

- PDF documents (resumes, certificates, etc.)
- Images (PNG, JPG, SVG)
- Any other downloadable assets

## File Naming Recommendations:

- Use lowercase letters
- Use hyphens instead of spaces
- Example: `nandani-raj-resume.pdf`
